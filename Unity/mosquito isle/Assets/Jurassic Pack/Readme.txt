    COMPSOGNATHUS Animated Compsognathus FBX model

    Thank you for purchasing this model. This asset contains two folders, dino folder and demo folder,
    dino folder only contains the FBX model with animations and textures.

    The "demo" folder is optional, there is a animation controller and a script allowing the dino to move 
    in all direction, play animation and sound.
    
    The MainCam script is only used to control the camera.
